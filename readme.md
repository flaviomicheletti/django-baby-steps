## Installation

Clone the project.

    cd your/projects/
    git clone https://github.com/devfuria/django-baby-steps.git
    cd django-baby-steps/

Create a virtual environment.
    
    python3 -m venv .venv && . .venv/bin/activate

To run

    python manage.py runserver
