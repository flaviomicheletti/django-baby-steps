## Installation

Clone the project.

    cd your/projects/
    git clone https://github.com/devfuria/django-baby-steps.git
    cd django-baby-steps/



Install Django.

    python -m pip install Django
    // or
    pip install Django==4.1

Testing...

    python3 -c "import django; print(django.get_version())"

    // or

    python3 ./step-000/foo.py
    # 4.1

When you finish

    deactivate
